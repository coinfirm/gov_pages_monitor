# gov_pages_monitor
Search weekly for:
- Crypto;
- Cryptocurrency;
- Crypto Currency;
- Virtual Currency;
- Bitcoin;
- Ethereum;
- Virtual asset;
- Blockchain;
- TRM;
- Distributed Ledger;
- Ripple;
- Stellar;
- Avalanche;
- Stable Coin;
# From:
- https://sam.gov/
- https://digital-strategy.ec.europa.eu/
- https://www.instantmarkets.com/
- https://ted.europa.eu
# Usage:
Scraps above pages using keywords in search section of the websites, if an article is found within 7 days with the keyword, title and link of this article is sent to slack chanel.
# Run:
python3.8 main.py
